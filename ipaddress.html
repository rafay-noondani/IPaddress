What is an IP address?

An **IP (Internet Protocol) address** is a unique numerical label assigned to every device connected to a computer network that uses the Internet Protocol for communication. Think of it as your device's mailing address on the internet. It serves two main functions: host or network interface identification and location addressing.

There are two types of IP addresses:

  • **IPv4:** The original version, using a format like `192.0.2.1`. These are limited and are being phased out.
  • **IPv6:** The newest version, designed to provide a massive number of unique addresses. The format is longer and more complex.

This tool helps you quickly identify your public IP address, which is the one that your home or office router uses to communicate with the rest of the internet. It can be useful for troubleshooting, setting up network devices, or simply satisfying your curiosity.
